# Journey Optimizer: Offer Decisioning en Edge

Offer Decisioning de Adobe Journey Optimizer se puede utilizar para personalizar y entregar ofertas en canales entrantes como sitios web, aplicaciones móviles, kioscos y otros puntos de contacto con los clientes.

El modelo para la personalización web y móvil mediante Offer Decisioning se encuentra en la sección del modelo Offer Decisioning de Recorrido del cliente [AQUÍ](../customer-journeys/offer_decisioning/offers-edge.md).
